It is a custom block that can change the position of the arranged "image" and "text" and can be displayed vertically on a smartphone.

== Description ==

The created custom block is

It's just a custom block with an image on the left and a textbox on the right, allowing you to change the image on the left and the textbox on the right.

In addition, the smartphone display is arranged vertically.

This is a slightly unusual form of the layout element "media and text".
